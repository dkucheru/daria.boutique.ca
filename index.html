Return Policy

At Daria.boutique.ca, we prioritize quality and strive to ensure that each item meets our high standards before it reaches you. Every item is carefully inspected and video-recorded prior to shipping to confirm its quality and condition.

Return Eligibility

We accept returns only under the following conditions:

  • Wrong Size Sent: If we mistakenly sent a different size than the one you ordered.
  • Wrong Color Sent: If we mistakenly sent a different color than the one you ordered.
  • Item Arrived Damaged: If the item arrives with visible damage, excluding issues caused by shipping.

Please Note:

We do not accept returns for any other reasons, including sizing or color perception. If you're unsure about sizing, please consult with our managers before placing your order.

Color variations may occur due to different screen displays. We cannot accept returns based on perceived color differences.

Return Process

  1. Contact Us: Notify us within 3 days of receiving your item with a description of the issue and a photo for verification.
  2. Return Authorization: If eligible, we will provide a return authorization with further instructions.
